프로그래머는 C언어 등의 언어로 만든 소스코드를 컴파일러로 이진수 체계인 기계어로 엮어냄.
오브젝트 파일을 묶는 링킹을 해서 실행 파일을 만든다.

오브젝트 파일: c언어 등의 언어로 된 소스코드를 컴파일해서 기계어로 적힌 파일

main 함수가 시작해서 끝날때까지 실행파일은 작동한다.

function(함수):기능
y=f(x)
x:매개변수

module(모듈):기능을 모은 것(헤더파일을 모은 것)
ex)stdio.h, stdlib.h

int func(int x)

y=func(x)

#include <stdio.h>(모듈을 불러오는 include 명령어)
헤더파일: 여러가지 function들이 있음
<>:파일명(<stdio.h>)
"":문자열(우리가 직접 만든 파일)

코드 블럭 사용 명령어

if/else
while
<> 특정 지칭
() 조건이 들어감(또는 입력값)
{} 실행해야할 로직의 코드를 넣음(코드 블럭의 단위)
"" 문자열
'' 문자 하나


int: 정수형
char: 문자 하나를 저장

+
-
*
/
%:나머지 연산자

대입연산자

오른쪽에 있는걸 왼쪽에 대입함
ex)a=a+1;

==: 등호

int 0;
00000000000000000000000000000000(32bit)

!:~이 아니다
!0;
=1

!= :같지 않을때

|: or(비트 연산자) ex) 2|1 (3)
&  ex)2&1(비트 연산자) (0)
|| (논리 연산자)
&& (논리 연산자) ex)2&&1(참)
<< :시프트 연산(왼쪽으로 시프트) ex) 10000 -> 00001000
>> :시프트 연산(오른쪽으로 시프트) ex) 10000 ->00100000
